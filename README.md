# Bike Unity - Приложение для местного туризма

Современное веб-приложение для изучения местных туристических достопримечательностей и событий.

## Возможности

- Современный, адаптивный интерфейс на React и Material-UI
- Интерактивная карта местных достопримечательностей
- Календарь и список событий
- Подробная информация о туристических местах
- Анимации и плавные переходы с использованием Framer Motion

## Технологии

- React
- TypeScript
- Material-UI
- Framer Motion
- Vite

## Требования

- Node.js (версия 14 или выше)
- npm (версия 6 или выше)

## Установка

1. Клонируйте репозиторий:
```bash
git clone [repository-url]
cd local-tourism-app
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Структура проекта

- `src/` - исходный код приложения
- `public/` - статические файлы и изображения
- `src/components/` - переиспользуемые компоненты
- `src/pages/` - страницы приложения
- `src/styles/` - стили и анимации
